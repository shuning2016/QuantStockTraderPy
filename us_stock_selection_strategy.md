# 美股超短线选股策略（2–5天持有期）

> 策略定位：每周一次执行，聚焦第一层板块筛选与第二层个股筛选，输出3–5只候选股供后续系统处理。

---

## 第一层：行业板块筛选

**执行时间：** 每周一  
**目标：** 从所有可选板块中筛选出本周强势板块 2–3 个

### 筛选指标（加权评分）

| 指标 | 判断标准 | 权重 |
|------|----------|------|
| 板块ETF相对强度 | 板块ETF 过去5个交易日涨幅高于 SPY（相对跑赢） | 35% |
| 资金净流入 | 板块ETF 近3日资金净流入连续为正（可用ETF成交量放大 + 价格上涨验证） | 30% |
| 近期催化剂 | 本周或下周内有明确行业事件：财报潮、政策发布、行业数据、产品发布 | 25% |
| 趋势方向确认 | 板块ETF 日线处于20日均线上方，排除处于下降通道的板块 | 10% |

### 排除规则

- 板块ETF 收盘价在20日均线下方 → **直接淘汰，不参与评分**

### 输出格式

```
板块名称 | 对应ETF | 综合得分（0–100） | 入选原因（1句话）
```

## 第二层：个股筛选

**执行时间：** 每周二  
**目标：** 从每个入选板块中筛选出 1–2 只候选股，合计输出 3–5 只

### Step A · 基本面门槛（硬性过滤）

以下条件全部为一票否决项，**不达标直接排除**：

| 条件 | 标准 |
|------|------|
| 市值 | > $5B |
| 日均成交量 | > 500万股 |
| 近期营收增速 | 最新季度 YoY > 10% |
| 机构持仓趋势 | 近一季度机构持仓增加或持平（排除机构大规模出逃的标的） |

### Step B · 技术面突破信号（满足4项中至少3项）

| 信号 | 判断标准 |
|------|----------|
| 均线位置 | 股价站上20日均线，且20日线斜率向上 |
| 放量突破 | 突破近期平台压力位，当日成交量 ≥ 20日均量的 1.5倍 |
| 相对强度 | 过去5日个股涨幅 > 板块ETF涨幅（板块内领涨） |
| MACD形态 | 日线 MACD 金叉，或 DIF 处于零轴上方且柱状图扩张 |
| K线形态（加分项） | 旗形整理后突破 / 杯柄形态 / 缩量回调至均线后放量启动 |

> K线形态为加分项，不计入3项达标统计，但同等分数下优先选择形态更优的标的。

### 输出格式

```
股票代码 | 所属板块 | 通过技术信号数 | 当前价格 | 关键支撑位 | 关键压力位
```

## 每周执行节奏

| 时间 | 任务 |
|------|------|
| 周一 | 执行第一层：扫描板块资金流向，评分排序，确定本周入选板块 2–3 个 |
| 周二 | 执行第二层：从入选板块中筛选个股，输出 3–5 只候选股交后续系统处理 |
| 周三–四 | 操作窗口（由后续系统处理） |
| 周五 | 复盘本周结果，优化评分参数 |

---

## 策略参数摘要

| 参数 | 值 |
|------|----|
| 市场 | 美股 |
| 持有周期 | 2–5个交易日 |
| 每周候选输出数 | 3–5只 |
| 板块筛选数量 | 2–3个 |
| 技术信号达标门槛 | 4项中满足3项 |
| 基本面过滤 | 硬性一票否决 |
